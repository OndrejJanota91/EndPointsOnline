How to manage your license

EndPoints integrates licensing system directly into Grasshopper. By hovering over the license button in top menu bar, for options will appear. Here you can activate/release your license for current computer. Delete license from current computer or get info about currently active users and license expiration.

  • Activate license: Open dialog for activation of your license on current computer. Two keys are needed for activation (license key and license token). If both are correct and there are free license avialbe, plugin will be activated after Validate button is clicked. Otherwise, message box will be shown with the information why license was not activated.
    If checkbox remember is checked, license will be saved on your computer and automatically activated next time Grasshopper is opened.

  • Release license: Release license from active usage, but keep license file save on your computer.
  • Delete license: Release license from active usage and delete license file from the computer if license was saved to you computer.
  • License info: Provides basic information about current usage of licenses and contact person, license key and ID and license expiration.

Plugin's license automatic activation during Grasshopper opening can tracked in the Rhino's command line. Activation, reactivation and deletion of your licenses via plugin top bar menu is comunicated by showing message windows.
